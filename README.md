# Learning_R
## Repo for learning R

### Whats in this Repo and how is it made
Will include different files for learning datastrucutures and statistical methods for calculating random numbers and functions, such as:

  1. Vector
  2. Matrix
  3. Data Frame

Will also include different build-in and self created datasets for experementing and machine learning approaches, such as:

R Files will be named by learning source or simply learning followed by a topic, e.g. *learning_vectors*
